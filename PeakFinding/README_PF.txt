RemoveOffset
MedianFilter
NoiseThreshold
Gaussian Filter
FindPeak
FindAllPeaksRefined
FindPeaksNoRingingRefined
QuadRefine
PeakAmplitudes
StandardAnalysis

The standard analysis method uses uses the other methods of the class with pre-established parameters 
The file also contains a PeakFindTemplate function to generate the peak template that will be used to unpile the signal
